# Davatar
Easily manage your ENS-based Web3 avatar!

### How do I add decentralized avatars to my Web3 app?
Use the [@davatar/react](https://www.npmjs.com/package/@davatar/react) library, it's easy!

## Local Development

Running app:
```bash
> npm install
> npm start
```
